# Delivery Directory Lister
Returns directory listings for S3 buckets.

```
NAME:
   delivery_dir_ls - delivery_dir_ls [options]

USAGE:
   delivery_dir_ls [global options] command [command options] [arguments...]

VERSION:
   1.1.0

AUTHOR(S):
   Jeremy Orem <oremj@mozilla.com>

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --addr ":8888"               Set the address on which to listen
   --bucket-prefix "net-mozaws-prod-delivery"   Sets S3 bucket prefix
   --help, -h                   show help
```
